# Introduce
This is a 'shell', to simulate a linux terminal
# Credential
Username is 'root', you never change it.
Password is blank, you shouldn't enter anything.
# Commands
shutdown: Shutdown Shell.
passwd: Change password.
reboot: Reboot Shell.
browser: Request a web page.
